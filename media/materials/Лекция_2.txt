Алгоритм работы
Инициализация переменных:

sum — сумма всех введенных чисел (изначально 0).

count — количество введенных чисел (изначально 0).

Первый ввод числа:

Пользователь вводит первое число.

Если это число 0, программа выводит ошибку:
"Ошибка: первое введенное значение — ноль. Невозможно вычислить среднее."
и завершает работу.

Цикл ввода чисел:

Пока введенное число не равно 0:

Добавить число к sum.

Увеличить count на 1.

Запросить следующее число.

Вычисление среднего:

Если count > 0, вычислить среднее по формуле:

average
=
sum
count
average= 
count
sum
​
 
Вывести результат:
"Среднее значение введенных чисел: <average>"
